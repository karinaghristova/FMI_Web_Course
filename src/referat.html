Какво е Chart.js?

Chart.js е популярна JavaScript библиотека с отворен код, която предоставя възможност за създаване на интерактивни графики и диаграми в уеб приложения. Тя е лека, проста за използване и предоставя високо стилизирани графики, които могат бързо да бъдат вградени в уеб страници. Освен това разполага с добре описана документация, която прави работата с нея изключително приятна. Chart.js е съвместима с всички известни JavaСcript библиотеки, като в това число се вклюват React, Vue и Angular.

Рендериране

За разлика от много други библиотеки, които използват SVG, Chart.js рендерира диаграмите на HTML5 платно, още известно като canvas. Това значително подобрява представянето, особено при използване на големи множества данни и комплексни визуализации. От друга страна, този начин на рендериране не позволява CSS стилизиране, което налага използването на вградени опции или създаването на персонализирани добавки (plugins), при желание за индивидуална стилизация.

Как да си инсталираме Chart.js?

  • Мениджър за Пакети Node - npm

    Първият начин, по който можем да инсталираме библиотеката Chart.js, е чрез мениджъра за пакети Node, като използваме командата npm install chart.js

  • Мрежа за доставка на съдържание - CDN

    Можем също така да използваме мрежа за доставка на съдържание, още известна като CDN. Необходимият линк може да бъде намерен тук

  • jsDelivr

    С този подход използваме скриптов елемент в HTML кода. За актуален линк, който се слага в него, може да посетите тази страница

  • Гитхъб

    Не на последно място, може да изтеглим последна версия на библиотеката Chart.js от Гитхъб.

Как се интегрира?

Интегрирането на Chart.js в нашия код може да стане с помощта чист JavaScript или чрез зареждане на модулни компоненти.

Пример за интеграция чрез скрипт (чист JavaScript)

                            <script src="path/to/chartjs/dist/chart.umd.js"></script>
                            </script>
                                 Your code
                            </script>

Пример за интеграция чрез включване на пакет

import Chart from 'chart.js/auto'
