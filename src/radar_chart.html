Радарна диаграма

Чрез този вид диаграми могат да бъдат показани множество от точки с данни и вариациите между тях. Полезни са, когато се цели сравняване на точки от две или повече различни множества от данни.

Пример

Тази диаграма използва вертикални стълбове за представяне на данните. Всяка стойност се представя от височината на стълба. Използва се за показване на тенденции в данните или за да се направи сравнение на множества от данни.

Пример

Диаграмата от примера представя оценките на Иван и Мария (сигурно съществуват) по различните езици, които изучават в училище (Това е стабилен езиков профил!!!). За целта се използват две множества от данни. (данните са измислени)

Етикети: ['Български', 'Английски', 'Немски', 'Испански', 'Френски']

Етикет на първото множество: 'Оценки на Иван'

Етикет на второто множество: 'Оценки на Мария'

Данните на двете множества са както следва:

Данни на първо множество: [3, 5, 2, 4, 6]

Данни на второ множество: [5, 4, 3, 2, 1]

Избрани са цветове за двете различни множества, техните точки, границите на техните точки и радиуса им. Указано е в опциите, че тиковете трябва да започват от 0 и легендата на диаграмата трябва да е под нея.

Примерен код за създаване на диаграмата може да намерите отдолу

Създаване на данни:

                        var chartData = {
                              labels ['Български', 'Английски', 'Немски', 'Испански', 'Френски'],
                              datasets [
                                {
                                  label: 'Оценки на Иван',
                                  data: [3, 5, 2, 4, 6],
                                  backgroundColor: 'rgba(255, 99, 132, 0.6)', 
                                  borderColor: 'rgba(255, 99, 132, 1)', 
                                  pointBackgroundColor: 'rgba(255, 99, 132, 1)', 
                                  pointBorderColor: '#fff', 
                                  pointRadius: 5, 
                                }
                                {
                                  label: 'Оценки на Мария',
                                  data: [5, 4, 3, 2, 1],
                                  backgroundColor: 'rgba(54, 162, 235, 0.2)', 
                                  borderColor: 'rgb(54, 162, 235)', 
                                  pointBackgroundColor: 'rgb(54, 162, 235)', 
                                  pointBorderColor: '#fff', 
                                  pointRadius: 5, 
                              
                                }
                              ]
                        };

Създаване на опции

                        var chartOptions = {
                              responsive: true;
                              scale: {
                                ticks: {
                                  beginAtZero: true
                                  max: 80
                                }
                              },
                              plugins: {
                                legend: {
                                  position: 'bottom'
                                }
                              }
                        };

Създаване на диаграмата

                        var radarChart = new Chart(ctx, {
                          type: 'radar',
                          data: chartData,
                          options: chartOptions
                        });
