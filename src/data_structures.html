Данни

Подаването на свойството/полето данни може да бъде осъществено под различни формати. По подразбиране свойството/полето данни на множеството данни се преобразува в съответствие с избраната диаграма. Важно е да се отбележи, че ако бъде използвано свойството/полето етикети, то трябва да съдържа същият брой елементи като този на множеството от данни с най-много елементи. Полето/свойството етикети се използва, за да бъдат поставени означения в оста на индексите, т.е. оста Х. За да бъдат рендерирани правилно етикетите, те трябва дая бъдат от тип стрин или число.

Масив от примитивни типове

                             const cfg = {
                               type: 'bar',
                               data: {
                                    datasets: [{
                                     data: [2, 3, 4, 5, 6]
                                    }],
                                    labels: ['слаб', 'среден', 'добър', 'много добър', 'отличен']
                                   }
                                 }

Масив от обекти

Това всъщност е форматът, който се използва вътрешно за преработените данни. В този режим преобразуването може да бъде изкллючено от опциите (parsing: flase). Ако това бъде направено, данните трябва да бъдат сортирани и представени в съответния формат, който определената диаграма и мащаби използват вътрешно. Нулевата стойност (null) може да бъде използвана за пропускане на стойности.

                             const cfg = {
                               type: 'bar',
                               data: {
                                    datasets: [{
                                     data: [{x: 'Ivancho', y: 5}, {x: 'Maria', y: 6}}]
                                    }],
                                   }
                                 }

Масив от обекти с персонализирани свойства/полета

Ако използваме пай или донът диаграма е важно обектът да има ключ, който да посочва стойността. Освен това, ако ключа съдържа точка, тя трябва да се избегне с двойна наклонена черта

                             const cfg = {
                               type: 'bar',
                               data: {
                                    datasets: [{
                                     data: [{brand: 'Mercedes', price: {value: 35}}, {brand: 'BMW', price: {value: 33}}]
                                    }],
                                   }
                               options: {
                                     parsing: {
                                      key: 'price.value'
                                    }
                                   }
                                  }

Обект

В този случай името на обекта служи за индекс, а стойността става стойност.

                             const cfg = {
                               type: 'bar',
                               data: {
                                    datasets: [{
                                     data: {
                                       Издържали изпита: 60
                                       Скъсани: 13
                                      }
                                    }],
                                   }
                                 }
