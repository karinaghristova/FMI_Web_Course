Точкова диаграма

Точковите диаграми наподобяват балонените диаграми, но тук елементите имат само 2 измерения и те определят тяхната локация. Тук е важно данните да се подават като обекти, които съдържат полета X и Y. Подходящи са за представяне на връзки между две променливи.

Пример

Диаграмата от примера показва как биха били разположени точките от едно измислено множество. (данните са измислени)

Етикет: 'Координати на точки'

Данни на множеството: [{x: 7, y: 15}, {x: 13, y: 27}, {x: 27, y: 21}, {x: 34, y: 16}, {x: 42, y: 17}]

Избрани е 1 цвят и е указано в опциите, че осите Х и Y са от линеен тип и се позиционират съответно отдолу и отляво.

Примерен код за създаване на диаграмата може да намерите отдолу

Създаване на данни:

                        var chartData = {
                              datasets [
                                {
                                  label: 'Координати на точки',
                                  data: [
                                      {x: 7, y: 15}, 
                                      {x: 13, y: 27},
                                      {x: 27, y: 21},
                                      {x: 34, y: 16},
                                      {x: 42, y: 17},
                                  ],
                                  backgroundColor: '#94618E', 
                                  borderColor: 'purple', 
                                  pointRadius: 6', 
                                  pointHoverRadius: 6', 
                                }
                              ]
                        };

Създаване на опции

                        var chartOptions = {
                              scales: {
                                x:  {
                                  type: 'linear'
                                  position: 'bottom'
                                },
                                y:  {
                                  type: 'linear'
                                  position: 'left'
                                }
                              },
                        };

Създаване на диаграмата

                        var scatterChart = new Chart(ctx, {
                          type: 'scatter',
                          data: chartData,
                          options: chartOptions
                        });
