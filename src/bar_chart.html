Колонна диаграма

Тази диаграма използва вертикални стълбове за представяне на данните. Всяка стойност се представя от височината на стълба. Използва се за показване на тенденции в данните или за да се направи сравнение на множества от данни.

Пример

Създаване на данни:

                        var chartData = {
                              labels ['Януари', 'Февруари', 'Март', 'Април', 'Май', 'Юни', 'Юли', 'Август', 'Септември', 'Октомври', 'Ноември', 'Декември'],
                              datasets [
                                {
                                  label: 'Похарчени пари през 2022',
                                  data: [578, 690, 896, 1296, 753, 432, 875, 321, 456, 497, 521, 685],
                                  backgroundColor: '#94618E', 
                                  borderColor: 'purple', 
                              
                                }
                                {
                                  label: 'Похарчени пари през 2023',
                                  data: [758, 960, 986, 673, 1321, 1294, 652, 895, 923, 657, 1106, 1213],
                                  backgroundColor: '#FFC0CB', 
                                  borderColor: '#F59FAE', 
                              
                                }
                              ]
                        };

Създаване на опции

                        var chartOptions = {
                              scales: {
                                y {
                                  beginAtZero: true
                                }
                              }
                        };

Създаване на диаграмата

                        var barChart = new Chart(ctx, {
                          type: 'bar',
                          data: chartData,
                          options: chartOptions
                        });
