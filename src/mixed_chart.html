Комбинирана диаграма

Тези диаграми позволяват създаването на комбинации от два или повече различни типа диаграми. Най-често се използват за смесване на колонна и линейна диаграма. При създаването на комбинирана диаграма е нужно да укажем типа на диаграмата за всяко множество от данни.

Пример

Диаграмата от примера представя информация за джобните, които група (нереални) ученици са изхарчили преди Коледа и след Коледа. За целта се използват две множества от данни. (данните са измислени)

Етикети: ['Яна', 'Фифо', 'Мирела', 'Алекс', 'Мая']

Етикет на първото множество: 'Изхарчени пари преди Коледа'

Етикет на второто множество: 'Изхарчени пари след Коледа'

Данните на двете множества са както следва:

Данни на първо множество: [30, 50, 20, 40, 60]

Данни на второ множество: [50, 30, 40, 20, 60]

Избрани са 2 цвята за двете различни множества и е указано в опциите, че стойностите на графиката по оста Y трябва да започват от 0, а легендата за диграмата трябва да се намира под нея.

Примерен код за създаване на диаграмата може да намерите отдолу

Създаване на данни:

                        var chartData = {
                              labels ['Яна', 'Фифо', 'Мирела', 'Алекс', 'Мая'],
                              datasets [
                                {
                                  label: 'Изхарчени пари преди Коледа',
                                  type: 'bar',
                                  data: [30, 50, 20, 40, 60],
                                  backgroundColor: 'rgba(75, 192, 192, 0.6)', 
                                  borderColor: 'rgba(75, 192, 192, 1)', 
                              
                                }
                                {
                                  label: 'Изхарчени пари след Коледа',
                                  type: 'line',
                                  data: [50, 30, 40, 20, 60],
                                  borderColor: 'rgba(255, 99, 132, 1)', 
                                  pointRadius: 5, 
                                }
                              ]
                        };

Създаване на опции

                        var chartOptions = {
                              scales: {
                                y {
                                  beginAtZero: true
                                }
                              },
                              plugins: {
                                legend: {
                                  position: 'bottom'
                                }
                              }
                        };

Създаване на диаграмата

                        var mixedChart = new Chart(ctx, {
                          type: 'bar',
                          data: chartData,
                          options: chartOptions
                        });
