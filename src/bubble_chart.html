Балонена диаграма

Балонената диаграма се използва, за да се покажат 3 измерения на данните. Първите 2 измерения определят локацията на балончето, а третото определя неговия размер.

Пример

Диаграмата от примера онагледява данни за координати на точки с различен радиус. (данните са измислени)

Етикет: 'Координати на точки'

Данни на множеството: [{x: 10, y: 20, r: 7 }, {x: 30, y: 7, r: 25 }, {x: 50, y: 47, r: 13 }, {x: 15, y: 31, r: 15 }, {x: 20, y: 40, r: 25 }, {x: 43, y: 23, r: 20 }, {x: 21, y: 20, r: 15 }, {x: 34, y: 40, r: 25 }, {x: 47, y: 13, r: 20 }]

Избран e 1 цвят и е указано в опциите, че стойностите на графиката по оста Y трябва да започват от 0.

Примерен код за създаване на диаграмата може да намерите отдолу

Създаване на данни:

                        var chartData = {
                              datasets [
                                {
                                  label: 'Координати на точки',
                                  data: [
                                      {x: 10, y: 20, r: 7 },
                                      {x: 30, y: 7, r: 25 },
                                      {x: 50, y: 47, r: 13 },
                                      {x: 15, y: 31, r: 15 },
                                      {x: 20, y: 40, r: 25 },
                                      {x: 43, y: 23, r: 20 },
                                      {x: 21, y: 20, r: 15 },
                                      {x: 34, y: 40, r: 25 },
                                      {x: 47, y: 13, r: 20 },
                                  ],
                                  backgroundColor: '#94618E', 
                                  borderColor: 'purple', 
                                }
                              ]
                        };

Създаване на опции

                        var chartOptions = {
                              scales: {
                                y {
                                  beginAtZero: true
                                }
                              }
                        };

Създаване на диаграмата

                        var bubbleChart = new Chart(ctx, {
                          type: 'bubble',
                          data: chartData,
                          options: chartOptions
                        });
